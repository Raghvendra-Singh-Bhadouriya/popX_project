# React + Vite

<!-- Landing Page -->
1. Created a two buttons on Landing page Create Account & LogIn.
2. If you have no account then you can click on Create Account then you redirect to the SignUp page.
3. if you have already an account then you can click on LogIn button then you redirect to the login page.
4. I am using useNavigate for redirecting between the components.

<!-- SignUp Page -->
1. In this page the input boxes already filled you don't need to fill it and you can direct on create account button then you will come on Profile page.
2. Here I also use useNavigate for redirect to the profile page.

<!-- LogIn page -->
1. In this page input box not filled but if you click on Login then you will come to the Profile page.
2. Here I also use useNavigate for redirect to the profile page.

<!-- Profile -->
1. Inside the profile page i used simple tags like h1, p, div, img and section.
2. In whole Proect i used Tailwindcss for styling.
